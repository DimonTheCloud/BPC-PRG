# Circles Project

## Co projekt dělá
Tento projekt pracuje se dvěma kružnicemi v rovině.

Program:
- spočítá vzdálenost mezi středy kružnic,
- určí, jestli se kružnice protínají,
- zjistí počet průniků (0, 1 nebo 2),
- umí kružnice vykreslit pomocí knihovny matplotlib.

## Struktura projektu
- `circle_stats.py` — výpočty a logika průniku
- `circles_intersection_draw.py` — vykreslení kružnic
- `circle_intersection.py` — hlavní spouštěcí skript
- `tests/` — testy
